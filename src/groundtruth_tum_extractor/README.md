# groundtruth_tum_extractor

This package exports ground-truth topics from `rosbag` files into TUM trajectory text files.

Supported inputs:

- `sensor_msgs/NavSatFix`
- `geometry_msgs/PoseStamped`

For the full English quick-start guide, build instructions, generic dataset usage, and launch examples, see the [repository root README](../../README.md).
